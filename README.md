# my-Stock Mobile

React Native Mobile application to monitor STOCKS

## Requirements

To run the project, please ensure you have the following installed:

- Long-term support version 14.18.0 or higher [Nodejs](https://nodejs.org/en/).
- Version 6.14.15 npm or higher.
- Expo Cli, run;

```bash
npm install --global expo-cli
```

## Installation

Clone the repository and switch branch from main to your local branch.

Run the following

```bash
npm install
expo install
```

## Usage

Run the project using expo cli command

```bash
expo start
```

If you don't want to use expo cli and want to develop react natively eject fom it using

```bash
expo eject
```

## Client Testing

Install [Expo Go](https://play.google.com/store/apps/details?id=host.exp.exponent&hl=en&gl=US) and paste `exp://exp.host/@vagenodipa/my-Stock?release-channel=default` to test the app.
